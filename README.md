# Chat with nouro flow

This example based on nouro flow assigned flows

To try the example you should have an account on Supabase 
With following content
```bash
create table
  public.credentials (
    id bigint generated by default as identity,
    user_id uuid null default gen_random_uuid (),
    openai_api_key text null,
    created_at timestamp with time zone not null default now(),
    constraint credentials_pkey primary key (id),
    constraint credentials_user_id_key unique (user_id),
    constraint public_credentials_user_id_fkey foreign key (user_id) references auth.users (id) on update cascade
  ) tablespace pg_default;

create table
  public.chats (
    uuid uuid not null default gen_random_uuid (),
    user_id uuid null default gen_random_uuid (),
    messages jsonb null,
    created_at timestamp with time zone not null default now(),
    constraint chats_pkey primary key (uuid),
    constraint public_chats_user_id_fkey foreign key (user_id) references auth.users (id) on update cascade
  ) tablespace pg_default;

create table
  public.bots (
    id bigint generated by default as identity,
    user_id uuid null default gen_random_uuid (),
    name text not null,
    uuid uuid not null,
    password text not null,
    created_at timestamp with time zone not null default now(),
    description text null,
    constraint bots_pkey primary key (id),
    constraint public_bots_user_id_fkey foreign key (user_id) references auth.users (id) on update cascade
  ) tablespace pg_default;
```

Required ENV variables:
```bash
# Public
PUBLIC_SUPABASE_ANON_KEY=""
PUBLIC_SUPABASE_URL=""
# Private
SUPABASE_DB_URL=""
SUPABASE_DB_KEY=""

NOURO_FLOW_API_KEY=""
```

Then run project
```bash
bun i && bun dev
```
